# Annotation

Annotation was conducted by four annotators (AWB, EM-W, J-PO, and KBN), who were provided with the relevant text (i.e., descriptions of item reporting) to answer each question. After the first round of annotation, the annotators raised questions about whether it was necessary to revisit the original full articles when the extracted text might be incomplete. Additional issues also emerged, such as how to select an option when the description was unclear. These issues were resolved through discussion, and the annotation guidelines were updated accordingly. The annotators were then asked to re-check all annotations labeled “Unclear” or “Not reported” using the full articles. CJV further supported this process by reviewing the annotations originally completed by Evan.

To complete the dataset for all 200 articles, we automatically assigned the option “Not reported” to articles with no evidence for a given question (e.g., when assessing whether a dataset was available or whether a factorial design involved a single factor).
